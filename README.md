# Hello world in C with i18n support with GNU gettext()

## Creating and Updating Locales

```bash
make locale
make edit-locale
make patch-locale
```

## Running Code  


```bash
make run
make run-default
```

## Running normally

```bash
LANGUAGE=en_US ./hello # Replace en_US with your required locale
```
